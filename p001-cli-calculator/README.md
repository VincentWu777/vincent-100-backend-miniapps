# P01 – CLI Calculator

A simple interactive command-line calculator designed to practice Python basics.

## Features
- Supports +, -, *, /
- Supports parentheses: (1 + 2) * 3
- Shows calculation history
- Graceful error handling
- Commands:
  - `history` – show past calculations
  - `quit` – exit the program

## How to Run

## Example
2 + 3 * 4
= 14

(1 + 2) * 5
= 15

history
